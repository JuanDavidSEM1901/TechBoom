# 🖥️ TechBoom - Tienda Online de Tecnología

Proyecto multimedia desarrollado como tienda virtual para la venta de productos tecnológicos, enfocado en brindar una experiencia de usuario moderna, intuitiva y eficiente.

---

## 🚀 Descripción del Proyecto

TechBoom es una plataforma web donde los usuarios pueden explorar, seleccionar y agregar productos tecnológicos a un carrito de compras, integrando elementos gráficos e interactivos.

---

## 🛠️ Tecnologías Utilizadas

- WordPress (CMS)
- Elementor (Constructor visual)
- HTML, CSS (estructura base)
- Diseño Responsive

---

## ⚙️ Características Principales

- Catálogo de productos organizado por categorías  
- Sistema de carrito de compras  
- Filtros por precio y productos destacados  
- Sección de contacto y formulario  
- Diseño adaptable a dispositivos móviles  

---

## 🧠 Metodología de Trabajo

Se utilizó la metodología ágil Kanban para la organización de tareas, permitiendo dividir el desarrollo en:

- Por hacer  
- En proceso  
- Finalizado  

Esto permitió un control eficiente del avance del proyecto.
